## Description

매번 초기세팅하기 힘들어서 만드는 nestjs boilerplate.

- cors 설정
- config, joi 설정 (validation)
- prettier 설정
- 예외 필터 설정
- Logger middleware 설정
- Jest 설정
- Dockerfile (기본)
- swagger 설정

!! .env gitignore에 추가해야함 !!

orm은 바뀔 수 있으니 설정하지 않음

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
